How it works

Source to protected in 3 steps

No servers to manage, no complex setup. Paste, compile, distribute.

01
📝

Add NXC_ macros

Insert NXC_ENCFUNC and NXC_ENCSTR into your Lua source. Passthrough stubs keep everything running locally.

02
⚡

Compile via dashboard or API

Upload source or POST to /v1/scripts/:id/compile. A unique randomized VM output is generated — no two builds ever identical.

Generate HWID-locked keys from the dashboard, Discord bot, or REST API. Users authenticate in under 50ms. Revoke and extend in real time.
